# Assignment-06 (Digitool Website)

This project is a product and cart based web application built with React. The main goal was to practice state management, component-based UI design, and basic cart functionality like adding and removing products.

## Project Overview

In this project, I built a product section where users can view items with details like name, price, and features. I also added a cart system where users can add or remove products and check the total price.

I tried to keep the design clean based on the given Figma layout and made the website work well on all screen sizes.

# Technologies Used

- React.js
- Tailwind CSS, DaisyUI
- JavaScript (ES6+)
- React-Toastify (NPM Package)
- JSON (for product data)


# Project Structure

I tried to keep the project well-organized:

- 'components' folder for reusable UI parts like Navbar and Product Card
- 'public' folder for static assets like product JSON data
- 'App.jsx' as the root component for managing state and logic


# Responsive Design

I made the website fully responsive using Tailwind CSS breakpoints (`sm`, `md`, `lg`) to ensure a consistent experience across mobile, tablet, and desktop devices.
